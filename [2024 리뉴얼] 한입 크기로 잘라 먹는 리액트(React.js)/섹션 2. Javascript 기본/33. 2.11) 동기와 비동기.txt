33. 2.11) 동기와 비동기

.동기
여러개의 작업을 순서대로, 하나씩 처리하는 방식.

기본적으로 자바스크립트는 동기적으로 실행이 된다!

단점이라면 만약 Task A는 0.1초, Task B는 10초, Task C는 0.2초 걸린다면 Task C는 Task B가 끝날때까지 기다려야한다.

치명적인 단점을 해소하고자 멀티쓰레드를 활용해야하나 자바스크립트 엔진은 1개의 쓰레드밖에 없음.

---

비동기란? 동기적이지 않다는 뜻, 즉 순서대로 처리하지 않는다!
